    Alle meine OpenSource-Projekte sind derzeitig auf GitHub gehostet. Mein GitHub Account findest Du hier : https://github.com/svenruppert

    Projectname: NanoVaadin

    URL: https://github.com/Nano-Vaadin-Demos

    Beschreibung:

    Bei den NanoVaadin Projekten handelt es sich um minimale Vaadin-Anwendungen die auf den Servletcontainern Jetty , Undertow und Meecrowave (Tomcat) aufsetzen. Ziel ist es hierbei, eine Lösung für kleine Projekte zu liefern die mittels fat-jar ausgeliefert werden sollen. Den Projekten liegen ebenfalls Docker-Files bei, die direkt eingesetzt werden können um damit ein Deployment durchzuführen.

    Wie man mit dem jeweiligen Projekt beginnen kann, ist in der beiliegenden README Datei beschrieben.
